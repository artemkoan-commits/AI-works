The Demon Slayer Fan Website is a modern, single-page, responsive web experience inspired by the anime Demon Slayer: Kimetsu no Yaiba. The design captures the dark fantasy atmosphere of Taishō-era Japan, blending traditional elements with a sleek, minimal digital interface.

🎯 Purpose

The website is created to:

Introduce users to the world and story of Demon Slayer
Showcase main characters in a structured and visual way
Provide an interactive gallery where users can upload and view images
Demonstrate front-end development skills using pure HTML, CSS, and JavaScript
🎨 Design Concept

The visual style focuses on:

Dark theme to reflect the anime’s tone
Crimson accents symbolizing danger, blood, and demon battles
Card-based layout for characters to keep content organized
Responsive grid system to ensure usability on both desktop and mobile

The layout is clean and intuitive, allowing users to scroll naturally through sections without distractions.

🧱 Structure

The website is divided into key sections:

Header & Navigation
Simple navigation for smooth scrolling between sections
Hero Section
Introduces the anime world with a large image and short description
About Section
Provides background story and setting
Characters Section
Displays main characters using cards with images and short descriptions
Gallery Section
Includes a dynamic image grid
Allows users to upload and display their own images instantly
Footer
Minimal closing section with copyright
⚙️ Functionality
Fully responsive design (mobile + desktop)
Image upload using JavaScript (FileReader API)
Dynamic gallery updates without page reload
No external libraries (pure front-end implementation)
💡 Concept Idea

This project represents a fan-based interactive showcase, combining storytelling and user interaction. It emphasizes simplicity, accessibility, and creativity while staying true to the aesthetic and emotional tone of the Demon Slayer universe.
